## Expansion Packs

### "Industrial Age" - Crafting System

Adds a 4-tier resource progression that gates advanced military units behind strategic resource management.

- **[CRAFTING.md](CRAFTING.md)** - Full crafting system specification
- 4 tiers: Base -> Refined -> Components -> Advanced Systems
- Industrial Sectors for manufacturing
- Crafting queue system

### "Shadow War" - Galactic Syndicate

Adds a criminal organization with contracts, black market, and comeback mechanics.

- **[SYNDICATE.md](SYNDICATE.md)** - Full syndicate specification
- 8-level trust progression
- Contract missions (4 tiers)
- Black Market for WMDs and components
- Recruitment mechanic for struggling empires
